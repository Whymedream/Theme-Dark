# Theme-Dark

Современная минималистичная тёмная тема для Discord с улучшенной читаемостью и чистым визуальным стилем.

---

## О проекте

**Theme-Dark** — это кастомная тема для Discord, предназначенная для использования через сторонние модификации клиента (например, BetterDiscord).

Тема перерабатывает стандартный интерфейс Discord, предлагая:

- Глубокую и сбалансированную тёмную палитру  
- Улучшенную контрастность текста  
- Аккуратные отступы и выравнивание  
- Минималистичный современный дизайн  

Основная цель проекта — создать удобный и визуально чистый интерфейс для ежедневного использования.

---

## Возможности

- Тёмная цветовая схема с оптимизированной контрастностью  
- Минималистичный и единый стиль интерфейса  
- Снижение визуального шума  
- Лёгкая настройка через CSS-переменные  
- Совместимость с актуальной версией Discord  

---

## Требования

Для работы темы требуется сторонняя модификация клиента Discord:

- BetterDiscord  

> Официальный клиент Discord не поддерживает пользовательские темы.

---

## Установка

1. Установите BetterDiscord с официального сайта.
2. Скачайте файл `Theme-Dark.theme.css` из данного репозитория.
3. Откройте Discord → **Настройки** → **Themes** → **Open Themes Folder**.
4. Переместите файл темы в открывшуюся папку.
5. Активируйте тему в разделе **Themes**.

---

## Настройка

Основные параметры можно изменить в блоке `:root` внутри файла темы:

```css
:root {
  --background-primary: #0f0f0f;
  --background-secondary: #141414;
  --accent-color: #5865F2;
}
